Global lexer

boost::parser::lexer

Synopsis

// In header: <boost/parser/lexer.hpp>

constexpr auto lexer;

Description

A variable template used to generate a lexer for use in token parsing. The resulting lexer has no associated tokens. Associate tokens with it by piping boost::parser::token_specs and/or boost::parser::token_charss after it.
