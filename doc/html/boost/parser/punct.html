Global punct

boost::parser::punct

Synopsis

// In header: <boost/parser/parser.hpp>

unspecified punct;

Description

The punctuation character parser. Matches the full set of Unicode punctuation clases (specifically, "Pc", "Pd", "Pe", "Pf", "Pi", "Ps", and "Po").
