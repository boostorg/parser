Global token_chars

boost::parser::token_chars

Synopsis

// In header: <boost/parser/lexer.hpp>

constexpr auto token_chars;

Description

Specifies one or more single-character tokens. Each character must be in the ASCII range (< 128), and must be of type char. If you want to specify tokens of longer than 1 character, use boost::parser::token_spec.
