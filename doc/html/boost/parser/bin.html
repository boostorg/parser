Global bin

boost::parser::bin

Synopsis

// In header: <boost/parser/parser.hpp>

constexpr parser_interface< uint_parser< unsigned int, 2 > > bin;

Description

The binary unsigned integer parser. Produces an unsigned int attribute. To parse a particular value x, use bin(x).
